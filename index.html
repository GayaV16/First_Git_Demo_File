git commands


git config --global user.email "emailid" -> To configure your github emailid
git config --global user.name "name" -> To configure your github username
git config --list -> To check configured github emailid and username along with keys
git clone -> To copy Remote (github) repo into local (laptop/pc)
cd -> To change directory
pwd -> To see present working directory
git ls -> To see only unhidden the files folder under pwd
ls -a -> To see hidden as well as unhidden the files folder under pwd
git status -> To check code status i.e modified, untracked, staged, or uptodate
git add . -> To all add files as it is ready to commit
git add filename -> To add specific file as it is ready to commit
git commit -m "Write message" -> To commit changes on git (this is were screenshots are taken and version control i.e changesin code is tracked by git)
git push origin main -> To push code on github
cd.. -> To come out of directory
mkdir Filename -> To create directory To copy file from local to remote
Step 1. git init -> To intialize a local repo on github Create repo on Github
Step 2. git remote add origin "Repolink" -> This will copy the stuff from local to the remote repo you created in github
Step 3 git remote -v -> To check in which remote repo we are currently
Step 4 git branch -> To check in which remote repo branch we are currently
Step 5 git branch -M main -> To rename remote repo branch name
Step 6 git add. -> To add code in remote repo
Step 7 git commit -m "write message" -> To commit code in remote repo
Step 8 git push origin main -> To push code in remote repo and code will be visible on github
git checkout branchname -> To switch branch
git checkout -b branchname -> To create new branch
git branch -d branchname -> To delete branch
git diff branchname -> To check difference
git merge branchname -> To merge 2 branches (But we do this using PR from git hub to avoid conflicts and some senior developer reviews and then merge the code)
git push -u origin main -> To set up stream so that we can just use git push instead of git push origin main
